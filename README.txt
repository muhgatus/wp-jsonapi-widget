Display Minecraft server information in your Wordpress with this simple Widget.

== Description ==

Simple lightweight widget to connect to a Bukkit Minecraft server running JSONAPI and show information about the server status.

Now suports multi langages!

== Installation ==

1. Upload the mc-jsonapi-widget folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the new widget in your sidebar and configure the IP and port of your JSONAPI within the Bukkit server. You also have to
add username, password and salt.


== Frequently Asked Questions ==

Q: It Won't connect?
A: Make sure you are using the correct ip-address, port, username, password and SALT for JSONAPI
Also if you had to open ports in your router or firewall for Bukkit, you may also need to repeat that process for the JSONAPI port.
